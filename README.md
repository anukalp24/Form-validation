
📖 Overview

This is a client-side Form Validation project built using HTML, CSS, and JavaScript.

The project validates user inputs before allowing form submission.
It ensures that users correct invalid fields step-by-step instead of submitting incorrect data.

🚀 Live Demo
🔗 form-validation-rho-steel.vercel.app
 
🚀 Features

✅ Validates empty fields

✅ Email format validation

✅ Password validation

✅ Shows error messages dynamically

✅ Removes previous errors before showing new ones

✅ Prevents form submission if validation fails

✅ Clean and responsive UI

🛠️ Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

DOM Manipulation

Event Listeners
